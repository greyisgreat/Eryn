About Eryn

Eryn v2.0 — voice-first desktop assistant. Runs entirely on your machine. No data collection. When the local server is running, she has full OS access: launch and kill apps, run shell commands, read/write any file, take screenshots, control volume, and more.

TAP OR SAY "HEY ERYN"

// ERYN v2.0 — CAPABILITIES

Always Available (Browser)

Voice recognition + speech synthesis
Open websites, web apps, and URLs in new tabs
Google, YouTube, Reddit, GitHub, Wikipedia search
AI-powered SVG diagrams (flowcharts, architecture, mindmaps)
Embedded Google Maps for any location
Read/edit local files via File System API (Chrome/Edge)
Push GitHub commits via API
Full conversation history across sessions
Hands-free mode with continuous listening

Launch any desktop app by name or voice
Kill/close any running application
Run shell commands and get output back
Read/write any file on the computer (not just browser sandbox)
Real-time CPU, RAM, disk, and process monitoring
Take screenshots of the full desktop
Read and write clipboard
Control system volume (Mac only, Windows needs nircmd)
Open URLs in the system default browser
